# AUTHENTICATION AND AUTHORIZATION

One of the basic fundamentals for building any app or service which requires storing personal data, handle trasactions or restrict features. The way for implementing these functions is authenticationa and authorization. 

## Authentication
Authentication is verifying whether someone is who they say they are. It is the process of identifying a user, device or system by validating the input credentials before granting the access to a network or its resources.
